HobbyBoss Custom Catalog Skin
About the site: This was a little weekend project to make simple and fast skin for the online catalog of the HobbyBoss model company. Their site is packed with useful information but it can be very slow, a pain to search, and is frequently down or inaccessible. This project is not associated with the HobbyBoss company at all (except it was made by a bit of a fan). While this remains a useful resource, it will be maintained here on GitHub
